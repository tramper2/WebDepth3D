# User Flow

## 1. 초기 진입 및 로딩 단계
1. 사용자가 웹 애플리케이션에 접속한다.
2. 화면 중앙에 "AI 모델을 불러오는 중입니다..." 라는 로딩 인디케이터가 표시된다. (최초 접속 시 모델 다운로드 캐싱 발생)
3. 로딩이 완료되면, 기본 제공되는 샘플 이미지가 복셀로 변환되어 화면에 나타난다.

## 2. 이미지 업로드 단계
1. 화면 하단(또는 상단)의 [이미지 업로드] 버튼을 클릭한다.
2. 파일 탐색기에서 이미지를 선택한다.
3. 시스템이 이미지를 256x256으로 크롭/리사이즈하고, 로딩 오버레이("깊이 분석 중...")를 띄운다.
4. AI 추론이 완료되면 새로운 3D 복셀 씬으로 부드럽게 전환된다.

## 3. 인터랙션 단계
1. **화면 제어:** 사용자가 마우스를 드래그하여 복셀 덩어리를 회전시키거나, 스크롤하여 줌인/줌아웃한다. (OrbitControls)
2. **뎁스 제어:** 화면 모서리에 있는 슬라이더(Depth Scale)를 좌우로 움직인다.
3. 큐브들이 앞뒤로 부드럽게 늘어나거나 압축되며 3D 입체감을 조절하는 시각적 피드백을 실시간으로 확인한다.
