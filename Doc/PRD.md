# Product Requirements Document (PRD): Web AI Voxelizer

## 1. 프로젝트 개요 (Project Overview)
본 프로젝트는 사용자가 업로드한 2D 이미지를 브라우저 환경에서 온디바이스 AI를 통해 깊이(Depth)를 추론하고, 이를 3D 복셀(Voxel) 형태의 인터랙티브 씬으로 변환하는 웹 애플리케이션이다. 서버 연산 없이 클라이언트의 리소스만으로 AI 추론과 3D 렌더링을 수행하는 실감형 미디어 경험을 제공한다.

## 2. 핵심 목표 (Goals)
- **온디바이스 AI:** `Transformers.js`와 `Depth Anything V2`를 활용하여 브라우저 내에서 즉각적인 뎁스 맵 추출.
- **렌더링 최적화:** 256x256 (65,536 픽셀) 해상도의 복셀을 60fps 프레임 드랍 없이 렌더링.
- **인터랙티브 경험:** 사용자가 슬라이더를 통해 복셀의 Z축 깊이감(Depth Scale)을 실시간으로 조작 가능.

## 3. 핵심 기능 명세 (Key Features)
1. **이미지 업로드 및 리사이징:**
   - 사용자가 로컬 이미지를 업로드.
   - 렌더링 및 추론 성능을 위해 내부적으로 캔버스를 이용해 256x256 해상도로 리사이즈.
2. **AI 깊이 추론 (Depth Estimation):**
   - 리사이즈된 이미지를 AI 모델에 입력하여 0~255 값의 깊이 데이터를 1D 배열로 반환.
3. **3D 복셀 변환 및 렌더링:**
   - 추출된 원본 컬러 픽셀 데이터와 뎁스 데이터를 1:1 매핑.
   - 각 픽셀을 3D 큐브(Voxel)로 변환하여 씬에 배치.
4. **실시간 뎁스 컨트롤 UI:**
   - Range 슬라이더 UI 제공.
   - 슬라이더 조작 시 CPU 재연산 없이 GPU 셰이더를 통해 즉각적으로 Z축 깊이 변화 반영.
5. **카메라 컨트롤:**
   - 마우스 드래그/줌을 통해 3D 복셀 씬을 다각도에서 관찰 가능.
